# IT Ресурсный батл

## Краткая версия правил

**Цель:**

2-6 команд соревнуются в выполнении максимального числа задач своего проекта, соревнуясь за специалистов и ресурсы.

- Каждый раунд: заявки на специалистов, переговоры, распределение, выполнение задач.
- Побеждает тот, кто наберет больше победных очков.
- Джокеры добавляют случайные события.
- В конце каждой игры — ретроспектива и репутационное голосование.

## Полная версия правил

### Общие положения
Команды (2-6) представляют IT-команды. Цель каждой — успешно реализовать как можно больше проектных задач, добывая специалистов через конкуренцию и переговоры, используя джокеры и стратегическое мышление.

### Компоненты:
- 6 уникальных проектов (описания и задачи для каждой команды)
- Пул специалистов для распределения между командами (программисты, тестировщики, дизайнеры, аналитики, девопсы, универсал)
- Таблица задач с нужными специалистами и баллами
- Колода джокеров

### Ход игры
1. **Подготовка:**
   - Каждая команда получает уникальный проект и роль: распределяют участников по ролям (менеджер на переговоры и др.).
   - Весь пул специалистов общий, задач больше, чем ресурсов.
2. **Раунд (спринт) — всего 4:**
   - **1 этап:** Тайно подать заявки на специалистов.
   - **2 этап:** Переговоры по спорным кандидатам (договариваются, идут торги, можно обмениваться ресурсами и задачами).
   - **3 этап:** Определяют, кто каких специалистов получил, распределяют их на задачи.
   - **4 этап:** Проверяют выполнение задач и отмечают результат.
   - В начале каждого 3 этапа разыгрывается джокер (событие).

### Механики и ограничения
- Некоторые задачи требуют конкретного набора специалистов (описано в карточках задач).
- Аналитики могут заменять часть тестировщиков, если задача требует нескольких.
- Если не удалось договориться о специалисте — из пула выбывает.
- Не договорились о задаче — штраф или задача "уходит".
- Джокеры влияют на ход игры (см. отдельный список событий).

### Система очков и победа
- Полная победа: выполнены все задачи — 20 очков.
- Частичная победа: выполнена ключевая + 2 любые задачи — 15 очков.
- Поражение: менее 50% задач или провал ключевой задачи — −20 очков.
- Допбонус команде с наибольшим числом выполненных задач: +3 очка.

### Ретроспектива
После игры проводится фасилитируемое обсуждение и репутационное голосование, чтобы команды отметили лучшие командные взаимодействия и союзников.
